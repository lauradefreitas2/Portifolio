# Laura de Freitas | Portfólio

Portfólio profissional reconstruído em Angular, com foco em backend Java/Spring, APIs, arquitetura e qualidade de entrega.

## Stack

- Angular 21
- TypeScript
- HTML semântico
- CSS responsivo
- Java 21 e Spring Boot como stack profissional destacada

## Rodar localmente

```bash
pnpm install
pnpm start
```

Abra:

```text
http://127.0.0.1:4200/
```

## Build

```bash
pnpm build
```
